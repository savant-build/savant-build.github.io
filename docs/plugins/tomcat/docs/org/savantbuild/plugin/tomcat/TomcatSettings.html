      • Property Detail

        • java.nio.file.Path binDirectory

          The directory that the Tomcat bin files for the current project are located. Defaults to src/main/tomcat/bin.

        • java.nio.file.Path buildDirectory

          The directory that the Tomcat tar.gz file is exploded to. Defaults to build.

        • java.nio.file.Path buildWebDirectory

          The directory that the Tomcat web application is located. Defaults to build/apache-tomcat-8.0.12/webapps/ROOT.

        • java.nio.file.Path confDirectory

          The directory that the Tomcat conf files for the current project are located. Defaults to src/main/tomcat/conf.

        • java.lang.String dependencyGroup

          The Tomcat dependency group that is used to fetch the Tomcat tar.gz file. Defaults to tomcat.

        • java.lang.String dependencyID

          The Tomcat dependency ID that is used to fetch the Tomcat tar.gz file. Defaults to org.apache.tomcat:apache-tomcat:8.0.12:tar.gz.

        • java.lang.String version

          The Tomcat version. This is used to fetch the tar.gz file and build Paths in this file. Defaults to 8.0.12.

        • java.nio.file.Path webDirectory

          The web application directory in current project. Defaults to web.
